# SCPSL-KillerCound [中文名:杀敌记录和死亡公告]

这个插件记录你的杀敌，杀敌会显示在称号上，死亡的时候会显示公告击杀者的名称

有BUG请联系QQ:1029482435 或 喵呜服务器寻找服主

# How to use [如何使用]

将插件丢进sm_plugins文件夹下然后启动
